**Diskussionsforum**

## Tekniker som används

- **Frontend**: React, React Router
- **Backend**: Express.js, better-sqlite3
- **Databas**: SQLite (via SQLiteStudio för hantering)
- **API**: RESTful endpoints

cd backend
npm install
node server.js
Backend körs på:
http://localhost:3000

cd frontend
npm install
npm run dev
Frontend körs på:
http://localhost:5173

Testdata
Testtrådar och svar finns i forum.db.
